# Neural Learning Behavior Laboratory

An interactive Streamlit-based application to visualize and compare classical neural network learning rules in 2D.

---

## 📌 Project Description

The **Neural Learning Behavior Laboratory** is an educational tool designed to demonstrate how different neural learning rules behave during training.

This project allows users to:

- Visualize decision boundaries
- Observe convergence behavior
- Compare learning rules
- Analyze hyperparameter sensitivity
- Study linear vs nonlinear separability

The goal is to provide conceptual clarity on how neural learning algorithms update weights and adapt over time.

---

## 🧠 Learning Rules Implemented

- **Perceptron Learning Rule**
- **Hebbian Learning Rule**
- **Logistic Regression (Gradient Descent)**
- **MLP (1 Hidden Layer with Backpropagation)**
- **Competitive Learning (Winner-Take-All)**

---

## 📊 Features

- 2D Decision Boundary Visualization
- Learning Curve Plotting
- Learning Rate Stress Test
- Experiment Logging & Comparison
- CSV Dataset Upload (Binary Classification)
- Automatic Numeric Feature Handling
- Accuracy Computation
- Weight Export

---

## 🧪 Built-in Datasets

- Linearly Separable Dataset
- XOR Dataset (Non-linearly Separable)
